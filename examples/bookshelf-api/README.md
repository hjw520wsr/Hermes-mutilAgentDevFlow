# 📚 Bookshelf API — Example Project

This is a placeholder for an example project built using the multi-agent workflow.

## What Would Be Generated

If you run:
```
> 用多 Agent 模式开发一个图书管理 REST API，用 FastAPI
```

The agents would create:
- `main.py` — FastAPI application entry point
- `models/` — SQLAlchemy/Pydantic models
- `routes/` — API route handlers (CRUD)
- `tests/` — Pytest test suite
- `requirements.txt` — Dependencies
- `README.md` — Auto-generated documentation

## Try It Yourself

```bash
# In Hermes Agent
> 用多 Agent 模式开发一个图书管理系统，技术栈 FastAPI + SQLite，放在 ~/projects/bookshelf-api
```
